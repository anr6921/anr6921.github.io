  • Home

  • Animations use CSS keyframes on load to fade into the page.

  • Page 1

  • Brick SVG background courtesy of Hero Patterns
  • Used CSS animations for consistent ball animations, JavaScript function called when button is clicked to drop red ball. While the CSS and JavaScript produce the correct motion graphics, unfortunetly it does not detect collisions at this time.

  • Page 2

  • Lightbulb SVG courtesy of Wikimedia Commons.
  • Lightswitch sound effects from Sound Jay
  • I changed the colors of the light-on and light-off in two separate SVG files. These two SVG files are in divs which change on click in a JavaScript function I wrote, making it feel like the user is turning the light on and off. This function also changes the background color, and makes a noise when the lightbulb is clicked.

  • Page 3

  • Bubble CSS couresty of Mark Bowley
  • I added a hover style to Bowley's CSS, changing the bubble's color and the cursor to a pointer. Addtionally, I added a "pop" JavaScript function so the bubbles would disappear when clicked. Popping a bubble also produces a sound effect.

  • Above and Beyond

  • I chose to go above and beyond by implementing sound effects into my project through JavaScript functions. Sound effects can be heard when turning the light on and off, as well as when you pop a bubble.
