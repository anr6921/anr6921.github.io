Hello, World!

Hello World is an open source game focused on teaching the basics of object oriented code. Created using C# in Monogame. I was the Creative Director for this project, and helped program the console window pop-up and validate user input.

Created in 24 hours for the 2017 WiC Hackathon, Hello World won two awards: Best Teaching Programming Hack sponsored by Hubspot, and Best Open Source Hack sponsored by Red Hat.

Gloom Box

Gloom Box is a 2D platformer created in Unity. It was produced by graduate student start up, Pentatonic Games, in Rochester, New York. My contribution to this project was part of an independant study to provide both conceptual and refined art assests to the Gloom Box team. My biggest contirbution to this project included the design and animation of NPC characters as well as UI/UX assets. Gloom Box won third place overall in RPI's 2017 GameFest.
